# CLAUDE.md
## Website design rules
- Make the UI modern and clean
- Prefer strong spacing and visual hierarchy
- Use larger headings and simpler sections
- Keep layouts uncluttered
- Make everything responsive
- Improve accessibility
- Reuse components instead of messy one-off styling
## If using Tailwind
- Prefer Tailwind utilities
- Avoid inline styles
- Use consistent padding, gaps, and rounded corners
- Use clear primary and secondary buttons
## When redesigning pages
- Keep the content
- Improve the design
- Do not break functionality
- Always make desktop and mobile versions look good
